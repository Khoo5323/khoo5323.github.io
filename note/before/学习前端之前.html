# 学习前端之前

# 三省吾身

以下为开放性问题，其中只是举例个别，思考时请往前深入一步，用解决问题的思维反问自己（不是为了劝退）

  1. 目前背景：学历、身份（学生、打工人）、专业、行业、岗位……
  2. 为什么学习前端：行业发展、兴趣、薪资……
  3. 对于前端岗位自己具备什么：保持学习，独立查找信息或答案、逻辑性……
  4. 学习之前做了什么准备工作：学习方式、学习资料、学习时间、辅助资源（行业内的人脉、前辈）……
  5. 学习方式：报班、自学（全职、兼职）、不同学习方式的区别、自己适合什么……
    1. 报班：学费、投入时间、依赖性、脱离了机构能否独立……
    2. 自学：自学能力、学习习惯、投入时间、学习资料……
      1. 全职：基本生活保障（无收入）、自律自控、遇到问题怎么解决……
      2. 兼职：投入时间、投入精力、态度（两手抓佛系、孤注一掷）……
